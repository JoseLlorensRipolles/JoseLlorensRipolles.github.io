Some of my latests projects

Snake-AI

Reinforcement Learning agent that learns to play Snake in the browser thanks to TensorflowJS. The agent uses a neural network and the Deep Q-Learning algorithm to beat you at snake. Website written with Angular and deployed in github-pages. Demo (and code) here.

Stockpile volume computation

Program to compute the volume of a 3D Point Cloud stockpile captured with a RGBD camera. I use mainly Open3D for 3D manipulation and scipy for its Delaunay triangularization. An small article can be read here.

Personal website

Personal website where you can get a glimpse of myself and my latests projects. This website has been built from scratch with Angular (v10) and Boostrap (v4). Example of recursion.

Advent of Code

Advent of code is a yearly challenge in which 25 puzzles are released from 1 to 25 of December. The puzzles can be solved in any programming language (I did it in Python) and cover a wide array of themes, data structures and algorithms.
